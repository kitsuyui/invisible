## Add noise

```console
$ invisible add-noise < example/plain.txt > example/noised.txt
```

It looks normal. But the size is grown from 446 bytes to 1070 bytes by noise.

## Encode

```console
$ invisible encode -m 'Hello, World!' < example/plain.txt > example/embeded.txt
```
